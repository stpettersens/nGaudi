What is nGaudi?

nGaudi is a platform agnostic build tool written in C# which runs on the Microsoft .NET or Mono platforms.

nGaudi can be thought of as being similar to Apache Ant in that it too also abstracts commands related to building software away from the operating system; but differs in that its build files are based on a JSON format rather than an XML format.

nGaudi is a port of the original Gaudi tool which was written for the JVM.

Features

Gaudi offers...
  • A compact, concise JSON based build file format.
  • Platform agnostic commands such as :erase.
  • Extendable commands via JSON based exension format.
  • A plug-in architecture to offer advanced functionality to the build system using the IronPython programming language.

Compatibility

Gaudi overall should be compatible with both the Microsoft .NET Framework 4+ or the latest release of Mono.
